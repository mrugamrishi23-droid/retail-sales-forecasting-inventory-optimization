\## 🚀 Project Overview



This project is an end-to-end \*\*Retail Analytics System\*\* that predicts future sales using time-series forecasting and optimizes inventory decisions using business-driven logic.



It helps retailers:

\- Predict future product demand 📈

\- Avoid stockouts and overstock situations 📦

\- Improve operational efficiency and profitability 💰



\---



\## 🎯 Problem Statement



Retail businesses often face:

\- ❌ Stock shortages (lost sales)

\- ❌ Overstocking (high holding costs)

\- ❌ Poor demand planning



This project solves these problems by combining:

\- \*\*Sales Forecasting (ARIMA Model)\*\*

\- \*\*Inventory Optimization (Reorder Point + Safety Stock)\*\*

\## ⚙️ Installation



\### 1. Clone Repository

git clone https://github.com/mrugamrishi23-droid/Retail-Sales-Forecasting-Inventory-Optimization



cd Retail-Sales-Forecasting-Inventory-Optimization




\### 2. Create Virtual Environment



Windows:

python -m venv venv

venv\\Scripts\\activate


Mac/Linux:

python3 -m venv venv

source venv/bin/activate



\### 3. Install Dependencies

pip install -r requirements.txt


\---



\## ▶️ Run the Project

python main.py
